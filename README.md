# Project - medibox
## 소개
- JNPMEDI의 [Maven Safety](https://www.jnpmedi.com/ko/maven-clinical-cloud/demo-videos/safety)을 모티브로 하였습니다.
- 사용자가 구매(복용)약을 등록, 관리하고 공공 API 활용하여 검색할 수 있습니다.
- 특히, 사용자가 입력 및 필요한 데이터를 빠르게 보여주기 위해 집중했습니다. 

<br/>

## 사용기술
- javaScript
- typeScript
- react.js
- next.js
